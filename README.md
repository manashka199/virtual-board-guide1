🔐 Вход и регистрация
Как войти в систему
1. Нажмите кнопку «🔐 Вход» в правом верхнем углу страницы
2. В появившемся окне введите логин и пароль из таблицы демо-аккаунтов
3. Нажмите кнопку «✅ Войти»
4. После успешного входа в меню появится ваше имя и новые кнопки:
   • «➕ Объявление» — создание новой публикации
   • «👤 Кабинет» — управление своими объявлениями
   • «⚙️ Админка» — только для администраторов
   • «🚪 Выход» — завершение сеанса
Как зарегистрировать нового пользователя
1. Нажмите «🔐 Вход» → «📝 Нет аккаунта? Зарегистрироваться»
2. Заполните все обязательные поля:
   • 📧 Логин: ваш email (будет использоваться для входа)
   • 🔑 Пароль: минимум 4 символа
   • 👤 ФИО: полностью, как в документах
   • 📞 Телефон: в формате +7 (999) 123-45-67
   • 🎓 Роль: выберите из списка (Ученик/Родитель/Учитель/Администратор)
3. Нажмите кнопку «✅ Войти»
4. Система автоматически:
   • Проверит уникальность логина
   • Создаст новый аккаунт
   • Откроет почтовый клиент с письмом администратору
   • Авторизует вас автоматически
5. Готово! Вы в системе 🎉
Как выйти из системы
1. Нажмите кнопку «🚪 Выход» в правом верхнем меню
2. Вы вернётесь в режим гостя
3. Все кнопки авторизованного пользователя скроются

Шаг 5: Готово!
• При отправке на модерацию — админ получит уведомление
• При сохранении в черновики — объявление видно только вам

⚠️ Важные правила заполнения полей
Заголовок: минимум 3 символа, должен содержать буквы русского алфавита, нельзя использовать повторяющиеся символы («аааа», «1111»)
Описание: должен быть связный текст с пробелами, минимум 50% символов должны быть буквами
Контакты: только корректный телефон (+7...), email, ссылка на Telegram (t.me/...) или ВКонтакте (vk.com/...)
Теги: вводятся через запятую, автоматически приводятся к нижнему регистру
Как редактировать объявление
1. Откройте «👤 Кабинет» в верхнем меню
2. Найдите объявление со статусом «Черновик» или «Отклонено»
3. Нажмите кнопку «✏️ Редактировать» рядом с объявлением
4. Внесите необходимые изменения в форму
5. Сохраните изменения:
   • «🚀 На модерацию» — снова отправит на проверку администратору
   • «💾 Черновик» — сохранит без отправки
Как удалить объявление
1. Найдите своё объявление в списке на главной странице или в личном кабинете
2. Нажмите кнопку «🗑️ Удалить» (кнопка видна только автору объявления и администратору)
3. Подтвердите удаление во всплывающем окне
4. Объявление будет удалено безвозвратно
🔍 Поиск и фильтрация объявлений
Поиск по ключевым словам
1. В поле «🔍 Поиск...» в верхней части страницы введите слово или фразу
2. Нажмите кнопку «🔎 Найти» или клавишу Enter
3. Система покажет объявления, где это слово есть в:
   • Заголовке
   • Тексте описания
   • Тегах
4. Для сброса поиска очистите поле и нажмите «Найти» снова
Фильтрация по типу объявления
1. В выпадающем списке «Тип» выберите нужный вариант:
   • 📋 Все типы — показать все объявления
   • ❓ Спрос — только объявления типа «Спрос»
   • ✅ Предложение — только объявления типа «Предложение»
   • 🔄 Обмен — только объявления типа «Обмен»
2. Нажмите кнопку «🔎 Найти» для применения фильтра
Фильтрация по категории
1. В левом боковом меню «📁 Каталог объявлений» выберите нужную категорию:
   • 📋 Все объявления
   • 📅 Расписание → Уроки / Мероприятия / Экзамены
   • 👨‍👩‍👧 Родителям → Собрания / Документы
   • 🎒 Ученикам → Домашние задания / Олимпиады
   • 🔄 Обмен/Продажа
   • 🔍 Потерянные вещи
   • 🤝 Волонтёрство
2. Список объявлений обновится автоматически
Фильтрация по тегам
1. В поле «Тег + Enter...» введите нужный тег, например «математика»
2. Нажмите клавишу Enter — тег появится как цветной чип с кнопкой удаления
3. Можно добавить несколько тегов для комбинированной фильтрации
4. Нажмите кнопку «🔎 Найти» для применения фильтров
5. Чтобы удалить тег — нажмите «×» на соответствующем чипе
6. Чтобы очистить все теги — нажмите кнопку «✕ Очистить»
Сортировка объявлений
• 📅 Сначала новые — свежие объявления отображаются вверху списка
• 📅 Сначала старые — старые объявления отображаются вверху списка

✅ Что работает в прототипе:

• Автоматическая отправка электронных писем
